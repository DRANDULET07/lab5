# Лабораторная работа №5: TypeScript в React

## Обзор проекта
Этот проект посвящен внедрению статической типизации в React-приложение для повышения надежности кода и раннего обнаружения ошибок.

## Выполненные задачи

### 1. Типизация компонентов и пропсов (Lab 5.1)
- **Интерфейсы данных**: Созданы интерфейсы `User` и `Skill` для строгого описания структуры объектов.
- **Union Types**: Использован тип `SkillLevel` (`'Beginner' | 'Intermediate' | 'Expert'`), чтобы ограничить возможные значения уровней навыков.
- **UserCard**: Реализован компонент с использованием `React.ReactNode` для пропса `children` и опциональным пропсом `isActive`.
- **SkillList**: Типизирован массив объектов для корректного отображения списка навыков с условной стилизацией.

### 2. Типизация состояния и событий (Lab 5.2)
- **Generics в useState**: Применена конструкция `useState<User[]>` для явного указания типа данных в массиве состояний.
- **Обработка событий**:
    - Для текстового поиска использован тип `React.ChangeEvent<HTMLInputElement>`.
    - Для кнопок использован тип `React.MouseEvent<HTMLButtonElement>`.
- **Логика поиска**: Реализована фильтрация пользователей по имени в реальном времени с обработкой случая "результаты не найдены".

## Как запустить
1. Установите зависимости: `npm install`
2. Запустите проект: `npm run dev`
